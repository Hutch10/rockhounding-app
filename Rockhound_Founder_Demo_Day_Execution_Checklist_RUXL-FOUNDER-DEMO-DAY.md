## Human-Readable: Founder Demo Day Execution Checklist (Ultra-Dense)

- Charge all devices, install final builds, reset demo data, verify OS/connectivity/backups

- Quiet, well-lit, screen sharing/recording, Wi-Fi/offline, backup power

### Pacing Cues

- Steady, unhurried, pause at transitions, allow audience reactions

### Confidence Anchors

- Recall past wins, affirm Rockhound's value, trust preparation

### Fallback Paths

- Predefined fallback per segment, quick data reset/relaunch, pre-recorded switch if needed

### Narrative Checkpoints

- Intro/context, mobile demo, desktop demo, cross-device sync, Q&A/closing

### Closing Cadence

- Summarize flows/outcomes, show call-to-action, invite feedback, thank/close

### Acceptance Criteria

- Devices/env ready, flows smooth/on-cue, confidence/pacing maintained, fallbacks ready, narrative checkpoints hit, demo closes with CTA/no critical issues
